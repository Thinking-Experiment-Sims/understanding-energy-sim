# Understanding Energy: Conservation & Work

Interactive "Energy Token" activity for 11th-grade physics. This simulation models the Bathtub analogy for energy conservation and the Work-Energy Theorem.

## Features
- **Interactive Post-It Notes**: Move 1J tokens between PE and KE.
- **Physical Ramp Model**: See the ball move in real-time based on energy distribution.
- **Work-Energy Phase**: Add or remove energy from the system using an external reservoir.
- **Guided Worksheet Integration**: 4 parts designed to match the "Understanding Energy" worksheet.

## Technologies
- HTML5 / CSS3 (Vanilla)
- Javascript (Vanilla)
- Canvas API

Built for [The Thinking Experiment](https://thinking-experiment-sims.github.io/interactive-physics/).
